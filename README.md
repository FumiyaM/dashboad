# dashboad
テストリポ

## オセロゲーム

このリポジトリにはオセロ（リバーシ）ゲームが含まれています。

### 機能
- 8x8のゲームボード
- 黒と白のターン制プレイ
- 有効手のハイライト表示
- 自動的なピースの反転
- スコア表示
- ゲームリセット機能
- レスポンシブデザイン

### 使用方法
1. `index.html`をブラウザで開く
2. 黄色にハイライトされたマスをクリックして手を打つ
3. ゲームが終了するまで交互にプレイ
4. 「リセット」ボタンで新しいゲームを開始

### ファイル構成
- `index.html` - メインのHTMLファイル
- `othello.css` - スタイルシート
- `othello.js` - ゲームロジック
